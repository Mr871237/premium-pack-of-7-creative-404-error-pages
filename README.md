## 🔍 Troubleshooting

### The page does not open

- Make sure you extracted the ZIP file first
- Open the correct HTML file
- Check that your browser is installed
- Try a different browser if needed

### Images or effects do not show

- Keep the file folder structure the same
- Make sure the CSS and JavaScript files are still in place
- Do not rename files unless you update the links inside the HTML

### The page looks broken

- Refresh the browser
- Check that all files stayed together in one folder
- Open the HTML file from the same folder as the CSS and JS files
