# Modern 404 Error Page

A modern, professional, and creative 404 error page built with HTML, CSS, and JavaScript.

## Features

- **Animated "Oops!"**: Each letter drops in with a smooth bounce, and the exclamation mark pulses for attention.
- **Premium Design**: Clean, elegant, and friendly look with glassmorphism, soft shadows, and rounded corners.
- **SVG Illustration**: A custom, modern illustration to reinforce the friendly tone.
- **Responsive**: Looks great on all devices, from mobile to desktop.
- **Clear Actions**: Two modern buttons for returning home or reporting a problem.

## File Structure

- `index.html` — Main HTML file for the 404 page.
- `style.css` — All styles, including layout, animation, and responsive design.
- `script.js` — Handles the letter drop animation and exclamation pulse.

## Usage

1. **Standalone**: Place these three files (`index.html`, `style.css`, `script.js`) in your web server's 404 error directory or configure your server to use this page for 404 errors.
2. **Integration**: You can copy the relevant code into your own project and adjust links as needed.

## Customization

- Change the SVG illustration in `index.html` for your brand or style.
- Edit the message or button links as needed.
- Adjust colors and fonts in `style.css` to match your branding.
