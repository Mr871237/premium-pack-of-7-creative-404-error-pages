# Modern 404 Error Page

A clean, professional, and responsive 404 error page for web projects.

## Features

- Minimal, modern layout with soft shadows and rounded elements
- Large, bold '404' text in a gradient
- Subheading: 'Page not found' with a short explanation
- Two action buttons: 'Go Home' (primary) and 'Contact Support' (outline)
- Neutral background with a subtle abstract SVG illustration
- Fully responsive: looks great on mobile, tablet, and desktop
- Built with [Tailwind CSS](https://tailwindcss.com/) and [Google Fonts](https://fonts.google.com/)

## Usage 

1. **Copy the files**
   - `index.html`
   - `styles.css` (optional, for custom styles)

2. **Open `index.html` in your browser** to preview the page.

3. **Integrate into your project**
   - Place `index.html` at your web root or configure your server to use it as the 404 error page.
   - Update the `Contact Support` button's email address as needed.

## Customization

- **Colors & Illustration**: Edit the SVG or Tailwind classes in `index.html` for your brand.
- **Text**: Change the main message, subheading, or button labels as you wish.
- **Buttons**: Link to your homepage, help center, or any other page.

## Technologies

- [Tailwind CSS CDN](https://cdn.tailwindcss.com/) for rapid styling
- [Google Fonts: Inter](https://fonts.google.com/specimen/Inter) for modern typography
- Custom SVG illustration (editable inline)
