# Modern 404 Page

A modern and animated 404 error page with responsive design and smooth animations.

## Preview
![404 Page Preview](preview.png)

## Features
- Modern design with glass effect
- Smooth animations
- Fully responsive
- Easy to customize

## Quick Start
1. Download these files:
   - `index.html`
   - `style.css`
2. Place them in your project
3. Open `index.html` to see the result

## Customize
### Colors
In `style.css`, change the colors:
```css
:root {
  --primary: #4f46e5;    /* Main color */
  --gradient-1: linear-gradient(135deg, #4f46e5 0%, #7c3aed 100%);  /* Gradient 1 */
  --gradient-2: linear-gradient(135deg, #3b82f6 0%, #2dd4bf 100%);  /* Gradient 2 */
}
```

### Text
In `index.html`, modify the text:
```html
<h1>Page Not Found</h1>
<p>Your message here</p>
```
