# 404 Error Page

This project contains a customized 404 error page with animated particle effects.

## How to Modify Content

### 1. Modify Text
To modify the page text, open the `index.html` file and modify the following elements:

- Page title: Modify the `<title>404 - Page Not Found</title>` tag
- Main error message: Modify the text in the `<h2 class="error-title">Oops! Page Not Found</h2>` tag
- Descriptive message: Modify the text in the `<p class="error-message">` tag
- Button text: Modify the text in the `<a href="/" class="home-button">` tag

### 2. Modify Style
To modify the page appearance:

1. Open the `css/styles.css` file
2. You can modify:
   - Colors
   - Fonts
   - Sizes
   - Margins and padding
   - Animations

### 3. Modify Particles
To modify the particle animation:

1. Open the `js/particles.js` file
2. You can adjust:
   - Particle speed
   - Particle size
   - Particle color
   - Number of particles
   - Particle behavior

## File Structure

```
├── 404.html          # Main page
├── css/
│   └── styles.css    # CSS styles
└── js/
    └── particles.js  # Particle animation
```

## Advanced Customization

For more advanced modifications, you can:

1. Change the font by modifying the Google Fonts links in the `<head>`
2. Add images or icons
3. Modify animation effects
4. Add additional interactions
Feel free to explore the CSS and JavaScript files for more advanced customizations!
