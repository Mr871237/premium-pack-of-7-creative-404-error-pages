# 404 Error Page

A modern, professional, and fully responsive 404 error page with animated background decorations, astronaut illustration, and premium UI/UX effects.

## Features
- Animated background with stars, planet, and abstract lines (SVG)
- Glassmorphism effect on the main container
- Neon and scan effect on the 404 title
- Animated astronaut illustration
- Professional SVG icons
- Responsive design for all devices
- Modern gradient button with glossy effect

## File Structure
- `index.html` — Main HTML file for the 404 page
- `style.css` — All styles and animations
- `script.js` — Handles the animated circles behind the astronaut
- `README.md` — This documentation

## Usage
1. Place all files (`index.html`, `style.css`, `script.js`) in the same directory on your web server or project.
2. Set your server or framework to use `index.html` as the 404 error page. For example:
   - **Apache**: Add `ErrorDocument 404 /path/to/index.html` to your `.htaccess` file
   - **Nginx**: Use `error_page 404 /path/to/index.html;` in your config
   - **Static hosting**: Set the 404 page in your provider's dashboard
3. Open a non-existent URL to see the 404 page in action.

## Customization
- **Colors**: Change the gradient and accent colors in `style.css` for your brand
- **Icons/Illustrations**: Replace SVGs in `index.html` for a different look
- **Texts**: Edit the messages in `index.html` to fit your tone

## Technologies Used
- HTML5
- CSS3 (Flexbox, Animations, Glassmorphism)
- JavaScript (vanilla, for animation only)
